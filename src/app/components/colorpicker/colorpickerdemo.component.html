ColorPicker ColorPicker is an input component to select a color.
ColorPicker can be displayed as an overlay (default) using the "inline" property.
ColorPicker can be displayed as inline by setting inline="true" property.
Default color format to use in value binding is "hex" and other possible values are "rgb" and "hsb".
